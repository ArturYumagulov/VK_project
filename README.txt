Программа для парсинга канала "Ярославль в курсе" в "В контакте".

Требования:
 - Python > 3.6

Для установки программы скопируйте сожержимое репозитория:
    - для Linux и Windows, MacOS (необходимо наличие установленной системы контроля версий на Вашем ПК (Git)):
        - "git clone https://github.com/Zico13/VK_project.git"
    - после клонирования репозитория активируйте виртуальное окружение
    = установите библиотеки из файла requirements.txt:
        - для Windows "pip install -r requirements.txt"
        - для Linux "pip3 install -r requirements.txt" аналогично для MacOS

Для использования программы необходимо проинициализировать класс VkParse() без аргументов,
для запуска сканирования
использовать метод класса VkParse: scaning() без аргументов.
Вывод программы: csv файл. Запишется в корне программы
